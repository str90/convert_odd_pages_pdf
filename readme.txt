Программа конвертирует произвольное количество PDF файлов. Извлекает нечетные страницы и делает новые PDF
файлы в папке result сохраняя иерархию каталогов.

Инструкция:
В папку source положить конвертируемые документы (можно в любой вариации вложенности каталогов и побочных файлов, 
отрабатываются только PDF файлы). Результат будет в папке result. 

The program converts variable amount of PDF files. Extracts odd pages and makes new PDFs in result folder
saving directory ierarchy.

Instruction:
Place PDFs in source folder (any directory ierarchy or side files, processes only PDFs). Final converted files will be 
in result folder.
